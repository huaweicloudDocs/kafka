# 查询可用区信息<a name="ListAvailableZones"></a>

## 功能介绍

在创建实例时，需要配置实例所在的可用区ID，可通过该接口查询可用区的ID。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ListAvailableZones)中调试该接口。

## URI

GET /v2/available-zones

## 请求参数

无
