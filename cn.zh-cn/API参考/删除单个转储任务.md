# 删除单个转储任务<a name="DeleteSinkTask"></a>

## 功能介绍

删除单个转储任务。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=DeleteSinkTask)中调试该接口。

## URI

DELETE /v2/\{project\_id\}/connectors/\{connector\_id\}/sink-tasks/\{task\_id\}
