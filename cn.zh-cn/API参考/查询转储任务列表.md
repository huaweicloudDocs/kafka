# 查询转储任务列表<a name="ListSinkTasks"></a>

## 功能介绍

查询转储任务列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ListSinkTasks)中调试该接口。

## URI

GET /v2/\{project\_id\}/connectors/\{connector\_id\}/sink-tasks
