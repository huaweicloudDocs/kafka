# 删除后台任务管理中的指定记录<a name="DeleteBackgroundTask"></a>

## 功能介绍

删除后台任务管理中的指定记录。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=DeleteBackgroundTask)中调试该接口。

## URI
