# 查询维护时间窗时间段<a name="ZH-CN_TOPIC_0171829479"></a>

## 功能介绍<a name="section51971645131316"></a>

## URI<a name="zh-cn_topic_0128036903_section747445721120"></a>

无。

**请求示例**

无。
