# 查询用户列表<a name="ShowInstanceUsers"></a>

## 功能介绍

查询用户列表。

Kafka实例开启SASL功能时，才支持多用户管理的功能。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowInstanceUsers)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/users
