# 批量删除用户<a name="BatchDeleteInstanceUsers"></a>

## 功能介绍

批量删除Kafka实例的用户

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=BatchDeleteInstanceUsers)中调试该接口。

## URI

```
}
```
