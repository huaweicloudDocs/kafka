# 查询分区指定时间段的消息<a name="ShowMessages"></a>

## 功能介绍

查询分区指定时间段的消息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowMessages)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/management/topics/\{topic\}/messages

