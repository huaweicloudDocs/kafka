# 批量添加或删除实例标签<a name="BatchCreateOrDeleteKafkaTag"></a>

## 功能介绍

批量添加或删除实例标签。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=BatchCreateOrDeleteKafkaTag)中调试该接口。

## URI

POST /v2/\{project\_id\}/kafka/\{instance\_id\}/tags/action
