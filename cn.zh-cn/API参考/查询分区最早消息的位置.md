# 查询分区最早消息的位置<a name="ShowPartitionBeginningMessage"></a>

## 功能介绍

查询分区最早消息的位置。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowPartitionBeginningMessage)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/management/topics/\{topic\}/partitions/\{partition\}/beginning-message
