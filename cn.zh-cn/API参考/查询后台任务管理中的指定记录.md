# 查询后台任务管理中的指定记录<a name="ShowBackgroundTask"></a>

## 功能介绍

查询后台任务管理中的指定记录。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowBackgroundTask)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/tasks/\{task\_id\}
