# 获取帐号名和帐号ID<a name="kafka-api-190128001"></a>

在调用接口的时候，部分URL中需要填入帐户名和帐户ID，所以需要先在管理控制台上获取到帐户名和帐户ID。帐户名和帐户ID获取步骤如下：

1.  登录管理控制台。

1.  单击用户名，在下拉列表中单击“我的凭证”。

    查看帐户名和帐户ID。
