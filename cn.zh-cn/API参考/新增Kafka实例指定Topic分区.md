# 新增Kafka实例指定Topic分区<a name="CreatePartition"></a>

## 功能介绍

新增Kafka实例指定Topic分区。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=CreatePartition)中调试该接口。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/management/topics/\{topic\}/partitions-reassignment
