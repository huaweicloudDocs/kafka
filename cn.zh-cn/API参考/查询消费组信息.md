# 查询消费组信息<a name="ShowGroups"></a>

## 功能介绍

查询消费组信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowGroups)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/management/groups/\{group\}
