# 查询用户权限<a name="ShowTopicAccessPolicy"></a>

## 功能介绍

查询用户权限。

Kafka实例开启SASL功能时，才支持多用户管理的功能。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowTopicAccessPolicy)中调试该接口。

## URI

GET /v1/\{project\_id\}/instances/\{instance\_id\}/topics/\{topic\_name\}/accesspolicy

## 请求示例

查询Topic的用户权限。

```
GET https://{endpoint}/v2/{project_id}/instances/{instance_id}/topics/{topic_name}/accesspolicy
```

## 响应示例

**状态码： 200**

查询成功。

```
{
  "name" : "topic-test",
  "policies" : [ {
    "owner" : false,
    "user_name" : "xxxa",
    "access_policy" : "pub"
  }, {
    "owner" : false,
    "user_name" : "root",
    "access_policy" : "all"
  } ],
  "topic_type" : 0
}
```
