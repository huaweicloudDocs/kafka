# 查询Kafka实例的协调器信息<a name="ShowCoordinators"></a>

## 功能介绍

查询Kafka实例的协调器信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowCoordinators)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/management/coordinators
