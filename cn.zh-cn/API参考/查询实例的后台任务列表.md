# 查询实例的后台任务列表<a name="ListBackgroundTasks"></a>

## 功能介绍

查询实例的后台任务列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ListBackgroundTasks)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/tasks
