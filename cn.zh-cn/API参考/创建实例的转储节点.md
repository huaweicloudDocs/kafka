# 创建实例的转储节点<a name="CreateConnector"></a>

## 功能介绍

## 调试

## 请求示例

```
POST https://{endpoint}/v2/{project_id}/instances/{instance_id}/connector

{
  "specification" : "100MB",
  "node_cnt" : "",
  "spec_code" : "kafka.c3.mini.connector"
}
```

## 响应示例

**状态码： 200**

创建转储节点成功。

```

}
```
