# 创建用户<a name="CreateInstanceUser"></a>

## 功能介绍

创建Kafka实例的用户，用户可连接开启SASL的Kafka实例。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=CreateInstanceUser)中调试该接口。
