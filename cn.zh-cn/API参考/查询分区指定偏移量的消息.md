# 查询分区指定偏移量的消息<a name="ShowPartitionMessage"></a>

## 功能介绍

查询分区指定偏移量的消息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=ShowPartitionMessage)中调试该接口。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/management/topics/\{topic\}/partitions/\{partition\}/message

```
GET https://{endpoint}/v2/{project_id}/instances/{instance_id}/manage/topics/{topic}/partitions/{partition}/message?message_offset={message_offset}
```
