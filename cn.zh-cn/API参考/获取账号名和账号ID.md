# 获取账号名和账号ID<a name="ZH-CN_TOPIC_0171829433"></a>

在调用接口的时候，部分URL中需要填入账号名和账号ID，所以需要先在管理控制台上获取到账号名和账号ID。账号名和账号ID获取步骤如下：

1.  登录管理控制台。
2.  单击用户名，在下拉列表中单击“我的凭证”。

    查看账号名和账号ID。
