# 设置用户权限<a name="UpdateTopicAccessPolicy"></a>

## 功能介绍

设置用户权限。

Kafka实例开启SASL功能时，才支持多用户管理的功能。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=UpdateTopicAccessPolicy)中调试该接口。

## URI

POST /v1/\{project\_id\}/instances/\{instance\_id\}/topics/accesspolicy

## 响应参数

无

## 请求示例

设置用户权限

```
POST https://{endpoint}/v1/{project_id}/instances/{instance_id}/topics/accesspolicy

{
  "topics" : [ {
    "name" : "topic-test",
    "policies" : [ {
      "user_name" : "root",
      "access_policy" : "all"
    } ]
  } ]
}
```
