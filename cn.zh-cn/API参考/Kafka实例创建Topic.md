# Kafka实例创建Topic<a name="CreateInstanceTopic"></a>

## 功能介绍

该接口用于向Kafka实例创建Topic。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=Kafka&api=CreateInstanceTopic)中调试该接口。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/topics
